# SOCMind-README
SOCMind

SOCMind is an advanced simulator for a Security Operations Center (SOC), designed to provide users with an interactive and applied work experience as a SOC analyst. The project enables real-time analysis of system logs and identification of suspicious activity, with an emphasis on simulating real-world scenarios for identifying cyber threats.

Key features:
* Intuitive user interface that includes registration, login, and session management.
* Uploading log files and automatic analysis to identify attack patterns.
* Advanced algorithms for identifying suspicious activities such as Brute Force attempts, unusual activity at night, and identifying unknown IP addresses.
* A professional simulation environment designed to train new analysts in the field of information security.

Project goal:
To provide a deep and practical understanding of monitoring and response processes for cyber threats, through an interactive platform that combines training and exposure to real-world scenarios in a controlled environment.

Technologies used:

Backend: Python

Databases: SQL for managing user data, logs and system information

Frontend: HTML, JavaScript, CSS
